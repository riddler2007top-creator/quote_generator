# Генератор случайных цитат
**Автор:** Румба Альбина

**Описание:**
Простое приложение для генерации случайных цитат из предопределённого списка. История сгенерированных цитат сохраняется в файл JSON и отображается в виде списка с возможностью фильтрации по автору и теме.

**Технологии:**
- Python 3.x.
- Tkinter (GUI).
- Random (случайный выбор).
- JSON (хранение данных).
- Git / GitHub.

## Как запустить приложение?
1. Установите Python 3.x.
2. Клонируйте репозиторий:
   ```
   git clone https://github.com/ВАШ_ЛОГИН/quote-generator.git
   ```
3. Перейдите в папку проекта.
4. Запустите:
   ```
   python main.py
   ```
5. Используйте интерфейс для генерации и просмотра цитат.
